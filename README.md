# Sensor Integrated 3D Printer Extrusion Monitoring System

This project explores the integration of sensors into an FDM 3D printer to monitor extrusion parameters and improve print quality through data-driven control.

## Technologies Used
- Load Cell
- Magnetic Encoder
- K-Type Thermocouple
- Arduino
- Machine Learning (planned)

## Project Overview
In FDM 3D printing, inconsistent filament extrusion can reduce print quality. This project aims to monitor key extrusion parameters using multiple sensors.

The collected data will be used to analyze extrusion behavior and eventually develop a machine learning model to optimize printing performance.

## Sensors Used
- Load Cell → measures extrusion force
- Magnetic Encoder → monitors filament movement
- K-Type Thermocouple → measures nozzle temperature

## Current Progress
- Sensor mounting and integration
- Data acquisition setup

## Planned Work
- Dataset generation from multiple prints
- Machine learning model training
- Development of closed-loop control for extrusion regulation

## Goal
To create a smart extrusion monitoring system capable of improving 3D printing reliability and quality.
